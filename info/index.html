Este dispositivo forma parte de la red River Sentinel, un sistema de vigilancia ambiental que monitoriza zonas de difícil acceso para detectar vertederos ilegales, acumulación de residuos y otras alteraciones del entorno natural.

¿Cómo funciona?

Detección continua

Vigila la zona de forma ininterrumpida, detectando cambios en el entorno que puedan indicar vertidos, acumulación de residuos u otras alteraciones ambientales.

Captura visual

Una cámara integrada toma fotografías periódicas de la zona, permitiendo identificar visualmente la aparición de residuos, vertederos ilegales o cualquier actividad sospechosa.

Transmisión de datos

Las imágenes y los datos recopilados se envían automáticamente a un sistema central, donde se procesan para detectar posibles incidencias de forma temprana.

Alertas y respuesta

Cuando se detecta una irregularidad, el sistema genera alertas para que el equipo responsable pueda intervenir a tiempo y evitar daños mayores al medio ambiente.

River Sentinel

Vigilancia Ambiental

River Sentinel es un proyecto de vigilancia ambiental que utiliza tecnología IoT para proteger el medio ambiente. Mediante estaciones autónomas ubicadas en zonas de difícil acceso, monitorizamos puntos vulnerables a la creación de vertederos ilegales y acumulación de residuos, contribuyendo a la detección temprana de estas actividades y a la conservación del entorno natural.

Por favor, no manipules este equipo

Esta estación contiene sensores y cámaras que trabajan de forma autónoma. Moverla, golpearla o alterarla puede comprometer la vigilancia de la zona y dificultar la detección de actividades contaminantes. Si observas algún daño o situación anormal, te agradeceríamos que nos lo comuniques.

Privacidad y protección de datos

Las imágenes captadas por este dispositivo se procesan internamente y nunca se almacenan ni se transmiten fuera de él. Solo se envían datos analíticos anonimizados, como la cantidad o el tipo de residuos detectados. Este sistema cumple con la normativa vigente de protección de datos y no recoge información personal de ningún tipo.
